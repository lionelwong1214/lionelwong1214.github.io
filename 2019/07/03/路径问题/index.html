路径问题

${pageContext.request.contextPath}用于解决使用相对路径时出现的问题，它的作用是取出所部署项目的名字。
①完全匹配：以“/”开头，以字母（非“*”）结束

如：上面提到的本项目中：/servlet/addBookServlet

②目录匹配：以“/”开头且以“/*”结尾

如：/test/*

/*

③扩展名匹配：以“*.”开头，以扩展名结束

如：*.do

④“/”用来表明对应的Servlet为应用默认的Servlet。在这种情况下Servlet路径是请求的URI去掉上下文路径并且路径信息为null。
